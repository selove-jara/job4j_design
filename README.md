# jobj4_design

## О проекте

Этот проект - в котором мы будем повышать свои знания программирования. 


###Темы разделов и описание:

1. Структуры данных и алгоритмы.

    - в этом разделе мы узнаем что такое Maven, Iterator, Generic,
    List, Set, Map и Tree
    - каждая из этим тем важна, ведь в будущем будем использовать 
    все перечисленное


2. Ввод-вывод -изучим взаимодействие с внешними ресурсами(интернет, файлы и тд).

    - здесь будем изучать разновидности ввода и вывода, Socket, логгирование и сериализация

3. SQL JDBC - база данных.
   
   - как говорится в первом же уроке "поздравляем" , ведь мы дошли до базы данных
    база данных будет использоваться очень часто, поэтому изучать стоит внимательно.

4. Профилирование приложения

    - Профилирование-это процесс изучения приложения для выявления проблем, 
     связанных с памятью или производительностью. 
    - При профилировании Java-приложения можно отслеживать виртуальную машину Java (JVM)
     и получать данные о производительности приложения, включая синхронизацию методов, 
     распределение объектов и сборку мусора.

5. ООД - дизайн
   
    - Объектно-ориентированный дизайн (ООД) — это методика проектирования, точка,
     в которой во время работы над продуктом сходятся все члены команды: дизайнеры, 
     проектировщики, разработчики, SEO-специалисты и копирайтеры.
   

Заключение: В каждом уроке дается информация и задания, но каждый должен обучаться дополнительно сам.
            Читать дополнительно уроки, смотреть видеоуроки по той теме что проходит, 
                или может что ему интересно или возникли трудности при выполнение
